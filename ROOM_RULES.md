## 1. 模式

| | Practice 练习 | Competitive 竞赛 |
|--|---------------|------------------|
| 密码轮换 | 20 分钟 | 12 分钟 |
| 每队每目标部署 | 不限 | 最多 3 次 |
| 央行闪灯间隔 | 每 **2 分钟** | 每 **4 分钟** |
| 适用 | 试讲、熟悉规则 | 正式比赛、排名 |

## 2. 身份

| 身份 | 权限 |
|------|------|
| **Player** | 攻外队州库、自愿参加央行 Bonus、为队挣队库 |
| **Spectator** | 观战地图、Activity、分队榜 / 个人榜 |
| **Host** | 模式、时长、Shuffle、进房密码、开始 / 结束比赛 |

## 3. 分队与地图

- 每队 **1～6 人**（房主可调上限，默认 4），**一队占一州**（西马 / 东马地图上一个基地）。
- **大厅流程**：进房默认临时名 `Player 1/2/3` → **确认显示名称**（仅个人名，不设队名）→ **点击编队空位**进驻某州；未进队者在 **候场区** 等待。
- **换位**：已进队者点「离队候场」→ 再点其他州 **空位** 进驻（v1 不做互换，只支持离队再进）。
- 房主可手动调整分队；**Shuffle 分队** / **随机重分州属** 规则不变。
- 地图上显示 **特工名字**；**余额只显示队库**（`team.vaultBalance`），无个人余额入账。

## 4. 经济总则：钱只进队库

| 来源 | 到账 |
|------|------|
| 攻外队州库（集体攻坚） | **攻者所属队库** |
| 央行 Bonus（阅读情报） | **完成者所属队库** |

- **分队榜**：按队库总额排名。
- **个人榜**：按「本人本场为队赢得的金额」排名（攻州 + Bonus 分别累计，展示合并总额）。

## 5. 管线 A — 攻某州 / 某队金库

### 5.1 流程

1. 在地图选 **外队**（不能打队友）。
2. **部署** → 系统 **随机抽 3 个小游戏**（从启用池）。
3. 三关全对 → **组合密钥确认** → **转账** → 金额进入 **你的队库**。

### 5.2 关卡池

| 玩法 | 维度 | 状态 |
|------|------|------|
| 拦截转账码 typing | 打字速度 | ✅ |
| 破译密语 dial | 逻辑 / 转盘 | ✅ |
| 潜入网格 grid | 记忆迷宫 | ✅ |
| 热力破解器 mastermind | 猜颜色排除法 | ✅ |
| 机器人密语 binary | 二进制 0–15 | ✅ |
| 量子同步 quantum | 时序记忆（3×3 闪烁复制） | ✅ |
| 数据熔炉 finale | 三关后校验（**随机**：经典 ①②③ 排序 **或** 斯特鲁普字体色排序） | ✅（非随机池） |
| 碎片排序 fragsort | （旧） | 已移出随机池 |
| 滤镜显影 cvfilter | （旧） | 已移出随机池 |
| 其他（钓鱼、日志…） | 按需启用 | 预留 |

- **练习 / 竞赛**：每次部署从启用池 **随机抽 3 个小游戏**，界面 **一次只显示当前一关**。
- **三关完成后**：最终校验（每局随机二选一）：**经典组合**（拖动片段 ①→②→③）或 **数据熔炉**（字写红/黄/绿，但字体颜色故意错位——**每局种子随机**本局目标颜色顺序，如 绿→黄→红；只看字体颜色，别看字）。
- **竞赛**：同队攻同一目标 → **相同种子、相同三关**；**每人独立失误**。
- 部署次数按 **队 × 目标** 共享。

### 5.3 夺取金额（固定基数）

- 每次成功攻外队州库：**名义 RM 250** 进入攻者队库（从受害队库扣除，扣不够则扣到 0）。
- **同队多人攻同一受害州**：每人独立完成三关与校验，**各算各的实拿**（不按人数分池）。
- **换目标、放弃任务、离开房间、改为观战** 会从当前攻坚名单移除。

### 5.4 失误扣款（攻坚专用）

整次部署三关的失误 **合计** 为一个计数：

```
实拿 = max(0, 250 − 失误次数 × 30)
实拿 = min(实拿, 受害队库余额)
```

- **9 次及以上失误** → 本次实拿 **RM 0**。
- 界面「实拿预估」会随失误实时更新。

**各关「1 次失误」定义（口播版）：**

| 关卡 | 算不算失误 | 说明 |
|------|------------|------|
| **typing** | ✅ 算 | 每错一字 / 一次有效错误 |
| **dial** | ❌ **不算** | 转盘操作不影响实拿 |
| **grid** | ✅ 算 | 踩雷、用完护盾等（按关卡提示） |
| **mastermind** | ✅ 算 | **用尽本局行数、谜题刷新** = 1 次；单纯猜错颜色 **只耗行，不加失误** |
| **binary** | ✅ 算 | 选错数字 = 1 次 |
| **quantum** | ✅ 算 | 复制序列时点错格子 = 1 次 |
| **finale 经典** | ✅ 算 | **排错并锁定** = 1 次 |
| **finale 熔炉** | ✅ 算 | 按错字体颜色顺序并锁定 = 1 次（别看字，看颜色；**目标顺序每局随机**） |
| **fragsort** | — | 已移出随机池 |

- 错满导致实拿归零后，仍可能占用部署次数。

## 6. 管线 B — 央行 Bonus

- 地图上 **央行图钉闪灯** → 自愿点击 → 阅读解密情报 → **3 道细节题**。
- 题目分两类标签：**EXTRACT · 细节验证**（常规）与 **ATTENTION · 序数/站点/时间/否定陷阱**（故意用第一站、旧时间、「不是」等误导）。
- 答错一题 **−RM 200**（从本波奖金池扣）；陷阱题答错会提示具体原因（如「问的是第二站，不是第一站」）。
- 题库含 **16 套**简报（含 4 套陷阱专练）；每波随机抽 1 套。
- 奖金进 **完成者队库**；同波同队仅 **首次全对** 入账（可重复练读）。

## 9. 观战、同步与进房密码

- 观战：地图、分队榜、个人榜、Activity、转账记录。
- **玩家端**：被盗时 Toast 显示攻击者与金额；HUD 红色 `-RM` 动画；地图金线 + 己方州闪红；侧栏战报（转账 / 破解）；战绩页高亮「你被攻坚」。
- 本地：`localStorage` 多标签可模拟多人（`server-config enabled: false`）。
- 多设备同房：**WebSocket 服务器**；断线后客户端 **自动重连**。
- **进房令牌**（可选）：房主在大厅开启并 **生成/刷新**；学员加入时输入令牌；服务器 **只存哈希**；已在房玩家与房主重连 **免验**。
- **历史赛局**：房主结束比赛后归档；教师可打开 `history.html` 查看。
